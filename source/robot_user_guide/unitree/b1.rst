1. Overview
===========
The B1 robot dog is an industrial grade, 12 DOF, quadruped robot.

2. Specifications
=================
.. list-table:: Technical Specifications
   :widths: 25 25

   * - Dimensions (Proning State)
     - 1202mm x 467mm x 297mm
   * - Dimensions (Standing State)
     - 1126mm x 467mm x 636mm
   * - Operating Temperature
     - 5 - 40 ℃
   * - Maximum traversal tilt
     - 30 degrees
   * - Maximum stair-climbing height
     - 20cm
   * - Maximum Speed
     - 1.2m/s	
   * - Charging Time
     - 1h - 2h
   * - Average Running Time
     - Continuous standing\: 2.5h, Continuous walking: 2h
   * - Weight
     - 50kg
   * - Rated Load
     - Standing\: 80kg, Walking: 20kg
   * - Protection Level
     - IP67 (IP68 optional)
   * - Motor
     - 12 x Servo Geared motors

3. Basic Operations
===================
The B1 can be operated in 2 ways, manual control using the included remote control 
or programmatically through Unitree's sdk.

Both methods are detailed in the user guides included in the :ref:`Resources<B1 Resources>` section below

:red:`Note: Unitree's SDK robot support is version specific, please check that you are using the correct version. Latest supported version for the B1 is v3.8.3`
