1. Overview
===========
The Scout Mini mobile robot is a four wheel differential drive platform. 


2. Specifications
=================

.. list-table:: Technical Specifications
   :widths: 25 25

   * - Steering
     - Differential steering
   * - Size
     - 612mm x 580mm x 245mm	
   * - Minimum Ground Clearance
     - 115mm
   * - Operating Temperature
     - -10 - 40 ℃
   * - IP Rating
     - IP22
   * - Maximum Speed
     - 3m/s	
   * - Maximum Angle of Tilt
     - Standard wheel\: <30° (unloaded), Mecanum wheel: <8° (unloaded)
   * - Charging Time
     - 2h
   * - User Power
     - 24V, 5A
   * - Weight
     - 22.5 - 23.5kg
   * - Rated Load
     - Standard wheel\: 10kg, Mecanum wheel: 20kg
   * - Motor
     - BLDC
