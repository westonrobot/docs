1. Overview
===========
The Hunter V2.0 mobile robot is an ackermann steering, 4-wheeled robot platform with use cases ranging from delivery to security inspection.

2. Specifications
=================

.. list-table:: Technical Specifications
   :widths: 25 25

   * - Steering
     - Ackermann steering
   * - Size
     - 980mm x 745mm x 380mm	
   * - Minimum Ground Clearance
     - 100mm
   * - Operating Temperature
     - -10 - 40 ℃
   * - IP Rating
     - IP22 (IP54 available upon request)
   * - Maximum Speed
     - 1.5m/s	
   * - Maximum Angle of Tilt
     - <10° (fully loaded)
   * - Encoder
     - Magnetic 2500ppr
   * - Charging Time
     - 3h (24V/30Ah), 6h (24V/60Ah)
   * - User Power
     - 24V, 10A
   * - Weight
     - 65-72kg
   * - Rated Load
     - 150kg
   * - Motor
     - 3 x 400W BLDC
