1. Overview

The LIMO PRO robot is equipped with an Nvidia Orin Nano processor running ROS on Ubuntu 20.04, serving as a fundamental platform for autonomous mobile robot research and education. Users have the option to integrate a manipulator (Mycobot 280 M5), allowing for precise mobile grasping to fulfill diverse task requirements
