1. Overview¶

The B1 robot dog is an industrial grade, 12 DOF, quadruped robot.

3. Basic Operations¶

The B1 can be operated in 2 ways, manual control using the included remote control or programmatically through Unitree’s sdk.

Both methods are detailed in the user guides included in the Resources section below

Note: Unitree’s SDK robot support is version specific, please check that you are using the correct version. Latest supported version for the B1 is v3.8.3
